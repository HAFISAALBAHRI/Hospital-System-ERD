**Project overview**:The Hospital Management System (HMS) is designed to streamline hospital operations by managing patients, doctors, departments, appointments, services, medical records, and billing.
The system ensures accurate record-keeping, efficient scheduling, and transparent billing, while remaining flexible for future enhancements.
 **Challenges faced**: Ambiguity in Requirements: Whether services are mandatory in appointments, whether doctors can manage multiple departments.
Price Change Handling: Deciding between updating service price globally vs storing historical price per usage.
Partial Payments: Designing billing to handle multiple transactions per appointment.
Department Head Flexibility: Whether head doctor is mandatory and if one doctor can lead multiple departments.
